# Isra University — OMR Console Pitch Deck

Stakeholder presentation for **OMR Console** (on-premises bubble-sheet exam processing) at Isra University, Hyderabad Sindh.

## Present

1. Open [`index.html`](index.html) in Microsoft Edge or Google Chrome
2. Press **F11** for fullscreen
3. Navigate with **←** **→** / **Space** · **N** for speaker notes · **P** to print/PDF

## Contents

| Path | Purpose |
|------|---------|
| `index.html` | Full slide deck (offline) |
| `SPEAKER_NOTES.md` | Talk tracks, demo script, Q&A |
| `README.md` | Presenter how-to |
| `assets/logo.png` | Isra University logo |
| `assets/screenshots/` | UI captures + analytics add-on mockups |

Works from a USB stick with no internet required.
