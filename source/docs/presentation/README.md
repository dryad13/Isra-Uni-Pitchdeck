# OMR Console — Stakeholder Presentation

Self-contained HTML slide deck for Isra University stakeholders.

## Open / present

1. Open `index.html` in **Microsoft Edge** or **Google Chrome** (double-click or drag into the browser).
2. Press **F11** for fullscreen.
3. Navigate:
   - **→** / **Space** / **Page Down** — next slide
   - **←** / **Page Up** — previous slide
   - **Home** / **End** — first / last slide
   - **N** — toggle speaker notes panel
   - **P** — print / save as PDF

## Export PDF

1. Press **P** (or File → Print).
2. Destination: **Save as PDF**.
3. Layout: Landscape; margins: None (or Minimum); enable “Background graphics”.

## Contents

| File | Purpose |
|------|---------|
| `index.html` | Full slide deck (offline-capable) |
| `assets/logo.png` | Isra University logo |
| `assets/screenshots/` | UI captures + add-on mockups |
| `SPEAKER_NOTES.md` | Talk tracks, Q&A, demo + add-on pitch |

Works from a USB stick with no internet required.
