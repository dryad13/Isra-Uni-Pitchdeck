# Source material

Reference docs and fixtures used to build the OMR Console stakeholder pitch deck. These are **not** required to present — open `../index.html` for the slides.

| Folder | Contents |
|--------|----------|
| `docs/` | Product bible, deployment, Canon setup, acceptance tests, design system, Windows build notes |
| `demo/` | Demo pack run instructions |
| `samples/` | Isra 150Q / 60Q sample notes and template JSON |
| `frontend-reference/` | App shell routes (`App.tsx`) used to map UI screenshots |
| `capture-fixtures/` | HTML mock screens + Playwright scripts used to generate screenshots |

Canonical product source remains in the OMR Console codebase (`spMDTA/windows`).
