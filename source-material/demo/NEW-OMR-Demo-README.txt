OMR DEMO
========

Copy this folder to the laptop (e.g. Desktop\OMR-Demo).
Do not run from the USB stick.

1. SETUP.bat     - run once (extracts the app)
2. START.vbs     - opens the demo in a desktop window
3. Close the OMR window when finished (STOP.bat is optional emergency stop)

If SETUP fails: make sure the whole folder was copied,
not just individual files.

Logs: OMR\data\omr.log
