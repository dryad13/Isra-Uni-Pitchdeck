# Isra 150Q — OMR POC Sample

Template family **150Q** (5 columns × 30 questions + 6-digit roll grid).

## Files

| File | Purpose |
|------|---------|
| `blank_template.png` | Blank high-res scan (2528×3507) used for calibration |
| `omr_marker.jpg` | Corner bullseye crop (120×120) for `CropOnMarkers`, cut from the high-res scan |
| `template.json` | OMRChecker field blocks (roll + 5 MCQ blocks), calibrated in `pageDimensions` space |
| `config.json` | Processing dimensions for this sample |
| `poc_layout_overlay.png` | Generated by POC — layout overlay on warped image |

## Run POC

```bash
# From repo root (requires OMRChecker runtime extras — see backend/omr_engine/requirements-vendor.txt)
source .venv/bin/activate
pip install -r backend/omr_engine/requirements-vendor.txt
python backend/scripts/run_poc.py
```

### Calibration notes

- Calibrated against the high-res flatbed scan (`imgtemp.jpeg`, 2528×3507 px).
- Marker centres detected at TL (148,122), TR (2324,122), BL (151,3327), BR (2323,3329);
  marker span ≈ 2173 × 3205 px (aspect 0.678).
- `pageDimensions` = **1080×1593** (preserves marker-span aspect). OMRChecker warps the
  marker-bounded region and resizes it to this space before reading bubbles, so all
  `origin`/`bubblesGap`/`labelsGap` values live in 1080×1593 coordinates.
- `bubbleDimensions` = 16×16 (measured bubble diameter ≈ 16 px in page space).
- Roll grid is a **horizontal** integer matrix (values 0-9 left→right, six digit rows
  top→bottom). The stock `QTYPE_INT` is vertical, so the roll block uses a new
  `QTYPE_INT_H` field type added to `omr_engine/src/constants.py`. Its name keeps the
  `QTYPE_INT` prefix so roll-block detection (`roll_number.py`, `pipeline.py`,
  `template_service._split_template`) still recognises it.
- Measured grid pitch: MCQ ≈ 27.2 px (A–D) / 27.1 px (rows); roll ≈ 27.0 px (0–9) / 26.9 px (digits).
- To re-tune visually, use OMRChecker `--setLayout` (set `PYTHONPATH` to `backend/omr_engine`):

```bash
cd samples/isra_150q
python -m src.entry --inputDir . --setLayout
```
